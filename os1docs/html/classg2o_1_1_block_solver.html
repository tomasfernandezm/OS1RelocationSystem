template<typename Traits >
bool g2o::BlockSolver< Traits >::setLambda ( double  lambda,
bool  backup = false 
)
virtual

update the system while performing Levenberg, i.e., modifying the diagonal components of A by doing += lambda along the main diagonal of the Matrix.

Note that this function may be called with a positive and a negative lambda. The latter is used to undo a former modification. If backup is true, then the solver should store a backup of the diagonal, which can be restored by restoreDiagonal()
